# QuerySense
A Smart Query Refinement and Prompt Engineering System using LLM and Human-in-the-Loop Interaction. 

# Overview

QuerySense is a Smart Query Refinement and Prompt Engineering System that helps users improve interactions with Large Language Models (LLMs). The system dynamically refines user input, generates optimized prompts, and supports Human-in-the-Loop refinement to improve output quality.

# Problem statement

Many users struggle to interact effectively with Large Language Models because they do not know how to write clear and structured prompts. Unclear or incomplete queries often result in inaccurate or irrelevant outputs, reducing the overall effectiveness of AI systems.

# Proposed Solution

QuerySense is a Smart Query Refinement and Prompt Engineering System that improves user input before it reaches the LLM. The system uses dynamic prompt engineering and Human-in-the-Loop refinement to generate optimized prompts, improve output quality, and make AI interaction more accessible to non-technical users.
